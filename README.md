# ⏳ Task Timer Project

A simple, efficient, and clean web-based Task Timer built using pure Vanilla technologies. This tool helps users track the time spent on specific tasks to improve productivity.

---

### 🚀 Features
*   **Start/Pause/Reset:** Full control over the timer status.
*   **Task Management:** Ability to name your task before starting the timer.
*   **Vanilla Logic:** Built entirely without external libraries or frameworks.
*   **Responsive Design:** Works perfectly on desktops, tablets, and mobile phones.
*   **Clean UI:** Simple and professional interface for a distraction-free experience.

---

### 🛠️ Technologies Used
*   **HTML5:** For the structural layout of the timer and input fields.
*   **CSS3:** For styling, including custom colors and responsive layouts.
*   **JavaScript (Vanilla):** For the core logic, interval handling, and DOM manipulation.

---

### 📂 How to Run
1.  **Clone or Download** this repository.
2.  Open the `index.html` file in any modern web browser.
3.  Enter your task name and start timing!
